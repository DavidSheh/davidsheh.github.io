гҖҢзҝ»иҜ‘гҖҚUnityдёӯзҡ„AssetBundleиҜҰи§ЈпјҲдёүпјү

еҸ‘иЎЁдәҺ 2017-08-21 жӣҙж–°дәҺ 2024-01-06 еҲҶзұ»дәҺ жҠҖжңҜзҝ»иҜ‘ йҳ…иҜ»ж¬Ўж•°пјҡ

жһ„е»әAssetBundles

еңЁAssetBundleе·ҘдҪңжөҒзЁӢзҡ„ж–ҮжЎЈдёӯпјҢжҲ‘д»¬жңүдёҖдёӘзӨәдҫӢд»Јз ҒпјҢе®ғе°ҶдёүдёӘеҸӮж•°дј йҖ’з»ҷBuildPipeline.BuildAssetBundlesеҮҪж•°гҖӮи®©жҲ‘д»¬жӣҙж·ұе…Ҙең°дәҶи§ЈжҲ‘д»¬е®һйҷ…дёҠеңЁиҜҙд»Җд№ҲгҖӮ
Assets/AssetBundlesпјҡиҝҷжҳҜAssetBundlesе°Ҷиў«иҫ“еҮәеҲ°зҡ„зӣ®еҪ•гҖӮжӮЁеҸҜд»Ҙе°Ҷе…¶жӣҙж”№дёәжүҖйңҖзҡ„д»»дҪ•иҫ“еҮәзӣ®еҪ•пјҢеҸӘйңҖеңЁе°қиҜ•жһ„е»әд№ӢеүҚзЎ®дҝқж–Үд»¶еӨ№е®һйҷ…еӯҳеңЁгҖӮ

BuildAssetBundleOptions

жңүеӨҡз§ҚдёҚеҗҢзҡ„BuildAssetBundleOptionsйҖүйЎ№еҸҜд»ҘйҖүжӢ©гҖӮзӣёе…ізҡ„еҗ„дёӘйҖүйЎ№еҸҜд»ҘеҸӮйҳ…и„ҡжң¬APIж–ҮжЎЈдёӯзҡ„е…ідәҺBuildAssetBundleOptionsзҡ„еҶ…е®№гҖӮ
иҷҪз„¶йҡҸзқҖйңҖжұӮзҡ„еҸҳеҢ–е’ҢеўһеҠ пјҢжӮЁеҸҜд»ҘиҮӘз”ұз»„еҗҲBuildAssetBundleOptionsйҖүйЎ№пјҢдҪҶжңүдёүдёӘзү№е®ҡзҡ„BuildAssetBundleOptionsжҳҜз”ЁжқҘеӨ„зҗҶAssetBundleеҺӢзј©пјҡ

  • BuildAssetBundleOptions.NoneпјҡжӯӨbundleйҖүйЎ№дҪҝз”ЁLZMAж јејҸеҺӢзј©пјҢиҝҷз§ҚеҺӢзј©жҳҜеәҸеҲ—еҢ–зҡ„ж•°жҚ®ж–Үд»¶зҡ„еҚ•дёҖеҺӢзј©зҡ„LZMAжөҒгҖӮ LZMAеҺӢзј©зҡ„ж–Үд»¶еңЁдҪҝз”ЁеүҚйңҖиҰҒеҜ№ж•ҙдёӘbundleи§ЈеҺӢзј©гҖӮиҝҷеҜјиҮҙжңҖе°Ҹзҡ„ж–Үд»¶еӨ§е°Ҹе’Ңз”ұдәҺи§ЈеҺӢзј©иҖҢзЁҚеҫ®еўһеҠ зҡ„еҠ иҪҪж—¶й—ҙгҖӮеҖјеҫ—жіЁж„Ҹзҡ„жҳҜпјҢеҪ“дҪҝз”ЁжӯӨBuildAssetBundleOptionsж—¶пјҢдёәдәҶдҪҝз”ЁжҚҶз»‘еҢ…дёӯзҡ„д»»дҪ•иө„жәҗпјҢеҝ…йЎ»йҰ–е…Ҳи§ЈеҺӢзј©ж•ҙдёӘжҚҶз»‘еҢ…гҖӮдёҖж—Ұbundleи§ЈеҺӢзј©еҗҺпјҢе°ҶдҪҝз”ЁLZ4еңЁзЈҒзӣҳдёҠйҮҚж–°еҺӢзј©пјҢLZ4еҺӢзј©еңЁдҪҝз”Ёbundleдёӯзҡ„иө„дә§ж—¶пјҢдёҚйңҖиҰҒжҸҗеүҚеҜ№ж•ҙдёӘbundleиҝӣиЎҢи§ЈеҺӢзј©гҖӮиҝҷжңҖйҖӮеҗҲз”ЁдәҺbundleдёӯзҡ„иө„дә§пјҢд»Ҙдҫҝд»ҺbundleдёӯдҪҝз”ЁдёҖдёӘиө„дә§е°Ҷж„Ҹе‘ізқҖжүҖжңүиө„дә§е°Ҷиў«еҠ иҪҪгҖӮжү“еҢ…и§’иүІжҲ–еңәжҷҜзҡ„жүҖжңүиө„жәҗйғҪжҳҜеҸҜиғҪдҪҝз”Ёзҡ„жҚҶз»‘еҢ…зҡ„дёҖдәӣзӨәдҫӢгҖӮз”ұдәҺиҫғе°Ҹзҡ„еҲқе§Ӣж–Үд»¶еӨ§е°ҸпјҢеҲ©з”ЁLZMAеҺӢзј©д»…жҺЁиҚҗз”ЁдәҺд»ҺејӮең°дё»жңәдёӢиҪҪиө„жәҗеҢ…гҖӮдёҖж—Ұж–Үд»¶иў«дёӢиҪҪпјҢе®ғдјҡзј“еӯҳдёәlz4еҺӢзј©еҢ…гҖӮ
  • BuildAssetBundleOptions.UncompressedAssetBundle: жӯӨbundleйҖүйЎ№д»Ҙж•°жҚ®е®Ңе…ЁжңӘеҺӢзј©зҡ„ж–№ејҸжһ„е»әbundleгҖӮжңӘеҺӢзј©зҡ„зјәзӮ№жҳҜиҫғеӨ§зҡ„ж–Үд»¶дёӢиҪҪеӨ§е°ҸгҖӮдҪҶжҳҜпјҢдёҖж—ҰдёӢиҪҪзҡ„еҠ иҪҪж—¶й—ҙдјҡжӣҙеҝ«гҖӮ
  • BuildAssetBundleOptions.ChunkBasedCompression: жӯӨbundleйҖүйЎ№дҪҝз”Ёз§°дёәLZ4зҡ„еҺӢзј©ж–№жі•пјҢиҝҷеҜјиҮҙжҜ”LZMAжӣҙеӨ§зҡ„еҺӢзј©ж–Үд»¶еӨ§е°ҸпјҢдҪҶдёҚеғҸLZMAйӮЈж ·еңЁдҪҝз”Ёд№ӢеүҚдёҚйңҖиҰҒж•ҙдёӘжҚҶз»‘еҢ…и§ЈеҺӢзј©гҖӮ LZ4дҪҝз”ЁеҹәдәҺеқ—зҡ„з®—жі•пјҢе…Ғи®ёе°ҶAssetBundleд»ҘеҲҮзүҮпјҲpiecesпјүжҲ–вҖңеқ—вҖқ(chunks)зҡ„еҪўејҸеҠ иҪҪгҖӮи§ЈеҺӢзј©еҚ•дёӘеқ—е…Ғи®ёдҪҝз”ЁеҢ…еҗ«зҡ„иө„дә§пјҢеҚідҪҝAssetBundleзҡ„е…¶д»–еқ—жңӘиў«и§ЈеҺӢзј©гҖӮ

дҪҝз”ЁChunkBasedCompressionдёҺжңӘеҺӢзј©зҡ„bundleе…·жңүеҸҜжҜ”иҫғзҡ„еҠ иҪҪж—¶й—ҙпјҢиҝҳе…·жңүеҮҸе°ҸзЈҒзӣҳеӨ§е°Ҹзҡ„йҷ„еҠ дјҳеҠҝгҖӮ

BuildTarget

BuildTarget.Standalone: еңЁиҝҷйҮҢпјҢжҲ‘д»¬жӯЈеңЁе‘ҠиҜүжһ„е»әз®ЎйҒ“пјҢжҲ‘д»¬е°ҶдҪҝз”ЁиҝҷдәӣAssetBundlesзҡ„зӣ®ж Үе№іеҸ°гҖӮ
жӮЁеҸҜд»ҘеңЁBuildTargetзҡ„APIзҡ„и„ҡжң¬еҸӮиҖғж–ҮжЎЈдёӯжүҫеҲ°еҸҜз”Ёзҡ„жҳҫејҸжһ„е»әзӣ®ж Үзҡ„еҲ—иЎЁгҖӮдҪҶжҳҜпјҢеҰӮжһңжӮЁдёҚеёҢжңӣеңЁжһ„е»әзӣ®ж ҮдёӯиҝӣиЎҢзЎ¬зј–з ҒпјҢеҲҷеҸҜд»ҘйҡҸж—¶еҲ©з”ЁEditorUserBuildSettings.activeBuildTargetпјҢиҝҷе°ҶиҮӘеҠЁжүҫеҲ°дҪ зӣ®еүҚзҡ„и®ҫзҪ®жһ„е»әе’Ңжһ„е»әAssetBundlesеҹәдәҺзҡ„зӣ®ж Үе№іеҸ°гҖӮ
дёҖж—ҰжӯЈзЎ®и®ҫзҪ®дәҶжһ„е»әи„ҡжң¬пјҢе°ұеҸҜд»Ҙжһ„е»әbundlesдәҶгҖӮеҰӮжһңжӮЁжҢүз…§дёҠиҝ°и„ҡжң¬зӨәдҫӢпјҢиҜ·еҚ•еҮ»Assets > Build AssetBundlesд»ҘеҗҜеҠЁиҜҘиҝҮзЁӢгҖӮ
зҺ°еңЁдҪ е·Із»ҸжҲҗеҠҹжһ„е»әдәҶAssetBundlesпјҢдҪ еҸҜиғҪдјҡжіЁж„ҸеҲ°дҪ зҡ„AssetBundlesзӣ®еҪ•жңүеҸҜиғҪжҜ”дҪ йў„жңҹжӣҙеӨҡзҡ„ж–Үд»¶гҖӮзЎ®еҲҮең°иҜҙпјҢжҳҜ2 *пјҲn + 1пјүдёӘж–Үд»¶гҖӮи®©жҲ‘д»¬иҠұдёҖзӮ№ж—¶й—ҙжқҘзңӢзңӢBuildPipeline.BuildAssetBundlesзҡ„дә§йҮҸгҖӮ

еҜ№дәҺдҪ еңЁзј–иҫ‘еҷЁдёӯжҢҮе®ҡзҡ„жҜҸдёӘAssetBundleпјҢдҪ е°ҶжіЁж„ҸеҲ°дёҖдёӘеҢ…еҗ«AssetBundle nameе’ҢAssetBundle name +.manifestзҡ„ж–Үд»¶гҖӮ

е°ҶдјҡжңүдёҖдёӘйўқеӨ–зҡ„bundleе’ҢmanifestпјҢе®ғдёҚдјҡдёҺжӮЁеҲӣе»әзҡ„д»»дҪ•AssetBundleе…ұдә«дёҖдёӘеҗҚз§°гҖӮиҖҢжҳҜд»Ҙе®ғдҪҚдәҺзҡ„зӣ®еҪ•пјҲAssetBundlesжһ„е»әеҲ°зҡ„зӣ®еҪ•пјүе‘ҪеҗҚгҖӮиҝҷжҳҜManifest BundleгҖӮжҲ‘д»¬е°ҶеңЁд»ҘеҗҺи®Ёи®әжӣҙеӨҡзҡ„еҶ…е®№д»ҘеҸҠеҰӮдҪ•дҪҝз”Ёе®ғгҖӮ

The AssetBundle File

иҝҷжҳҜзјәе°‘.manifestжү©еұ•еҗҚзҡ„ж–Үд»¶пјҢд»ҘеҸҠдҪ еңЁиҝҗиЎҢж—¶еҠ иҪҪзҡ„еҶ…е®№пјҢд»ҘеҠ иҪҪдҪ зҡ„иө„дә§гҖӮ
AssetBundleж–Үд»¶жҳҜдёҖдёӘеңЁеҶ…йғЁеҢ…еҗ«еӨҡдёӘж–Үд»¶зҡ„еӯҳжЎЈгҖӮжӯӨеӯҳжЎЈзҡ„з»“жһ„еҸҜиғҪдјҡзЁҚжңүеҸҳеҢ–пјҢе…·дҪ“еҸ–еҶідәҺе®ғжҳҜAssetBundleиҝҳжҳҜеңәжҷҜAssetBundleгҖӮиҝҷжҳҜдёҖдёӘжӯЈеёёзҡ„AssetBundleзҡ„з»“жһ„пјҡ

еңәжҷҜAssetBundleжӣҙж”№иҮӘж ҮеҮҶзҡ„AssetBundlesпјҢеӣ дёәе®ғй’ҲеҜ№еңәжҷҜеҸҠе…¶еҶ…е®№зҡ„жөҒеҠ иҪҪиҝӣиЎҢдәҶдјҳеҢ–гҖӮиҜҘеӣҫеғҸжҳҫзӨәеңәжҷҜbundleзҡ„еҶ…йғЁз»“жһ„пјҡ

The Manifest File

еҜ№дәҺз”ҹжҲҗзҡ„жҜҸдёӘеҢ…пјҢеҢ…жӢ¬йҷ„еҠ зҡ„жё…еҚ•еҢ…пјҢйғҪдјҡз”ҹжҲҗе…іиҒ”зҡ„жё…еҚ•ж–Үд»¶гҖӮжё…еҚ•ж–Үд»¶еҸҜд»ҘдҪҝз”Ёд»»дҪ•ж–Үжң¬зј–иҫ‘еҷЁжү“ејҖпјҢе№¶дё”еҢ…еҗ«иҜёеҰӮеҫӘзҺҜеҶ—дҪҷж ЎйӘҢпјҲCRCпјүж•°жҚ®е’ҢеҢ…зҡ„дҫқиө–жҖ§ж•°жҚ®зҡ„дҝЎжҒҜгҖӮеҜ№дәҺжӯЈеёёзҡ„AssetBundlesпјҢд»–д»¬зҡ„жё…еҚ•ж–Үд»¶е°ҶеҰӮдёӢжүҖзӨәпјҡ

иҝҷе°ҶжҳҫзӨәAssetBundlesеҰӮдҪ•е…іиҒ”д»ҘеҸҠе®ғд»¬зҡ„дҫқиө–е…ізі»гҖӮзҺ°еңЁпјҢеҸӘиҰҒжҳҺзҷҪиҝҷдёӘbundleеҢ…еҗ«дәҶAssetBundleManifestеҜ№иұЎпјҢиҝҷеҜ№дәҺжүҫеҮәеңЁиҝҗиЎҢж—¶еҠ иҪҪе“ӘдёӘbundleдҫқиө–жҳҜйқһеёёжңүз”Ёзҡ„гҖӮиҰҒдәҶи§Јжңүе…іеҰӮдҪ•дҪҝз”ЁжӯӨbundleе’ҢmanifestеҜ№иұЎзҡ„жӣҙеӨҡдҝЎжҒҜпјҢиҜ·еҸӮйҳ…дҪҝз”Ёжң¬ең°AssetBundlesзҡ„ж–ҮжЎЈгҖӮ
